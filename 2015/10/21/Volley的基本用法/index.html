Volley的基本用法解析

发表于 2015-10-21 | 分类于 Android

# Volley简介


上图是在Google I/O 2013大会上演讲的配图。这张图片概括了Volley的特点：适合频繁请求，数据量小的情况，而大数据，如下载，流媒体则不适用。Volley可是说是把AsyncHttpClient和Universal-Image-Loader的优点集于了一身，既可以像AsyncHttpClient一样非常简单地进行HTTP通信，也可以像Universal-Image-Loader一样轻松加载网络上的图片。

#准备工作

使用Volley需要下载它jar包导入到工程中。

1.用Git命令:
git clone https://android.googlesource.com/platform/frameworks/volley

2.CSDN下载地址：
http://download.csdn.net/detail/sinyu890807/7152015

#StringRequest用法

public class MainActivity extends AppCompatActivity {

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    //请求队列
   RequestQueue requestQueue = Volley.newRequestQueue(this);
    StringRequest stringRequest = new StringRequest("http://www.baidu.com",
            new Response.Listener<String>() {
        public void onResponse(String s) {
            System.out.println(s.toString());
        }
    },new Response.ErrorListener(){
        @Override
        public void onErrorResponse(VolleyError volleyError) {
            System.out.println("error");
        }
    });
     //把stringRequest加入请求队列
    requestQueue.add(stringRequest);
}

}

1.首先拿到一个请求队列。

2.new 出StringRequest对象，StringRequest的构造函数需要三个参数，第一个是请求目标的URL地址，第二个是响应成功的回调，第三个是响应失败的回调。

3.把StringRequest加入到请求队列。

注意：使用Volley是需要加网络权限的，所以要在AndroidManifest.xml添加如下权限：

<uses-permission android:name="android.permission.INTERNET" />

# Volley加载图片

public class MainActivity extends AppCompatActivity {
private ImageView imageView;

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    imageView = (ImageView) findViewById(R.id.imageView);
    loadImageVolley();
}

public void loadImageVolley(){
    String imageUrl = "http://i11.tietuku.com/10250403e075156a.jpg";
    RequestQueue requestQueue = Volley.newRequestQueue(this);
    //使用LruCache即缓存，可以提高性能，避免程序OOM
    final LruCache<String,Bitmap> lruCache = new LruCache<>(20);
    ImageLoader imageLoader = new ImageLoader(requestQueue, new ImageLoader.ImageCache() {
        @Override
        public Bitmap getBitmap(String s) {
            return lruCache.get(s);
        }

        @Override
        public void putBitmap(String s, Bitmap bitmap) {
            lruCache.put(s, bitmap);
        }
    });
    ImageLoader.ImageListener listener = imageLoader.getImageListener(imageView,R.mipmap.ic_launcher,R.mipmap.ic_launcher);
    imageLoader.get(imageUrl, listener);

}
}

1.这里同样需要先创建一个请求队列。

2.使用LruCache即缓存。

3.new出ImageLoader对象 

public ImageLoader(RequestQueue queue, ImageLoader.ImageCache imageCache)

ImageLoader的构造方法需要两个参数，一个是RequestQueue队列，第二个是ImgaeCache，new出来需要实现两个方法。

# NetworkImageView的用法

NetworkImageView也是用来加载图片的，跟上面不同的是，它是自定义控件的。

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
xmlns:tools="http://schemas.android.com/tools" android:layout_width="match_parent"
android:layout_height="match_parent"
android:orientation="vertical"
tools:context=".MainActivity">
<ImageView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:id="@+id/imageView"/>
<com.android.volley.toolbox.NetworkImageView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:id="@+id/imageView1"/>
    />

</LinearLayout>

自定义控件继承自ImageView,接着在MainActvity中获取控件，进行设置。

public class MainActivity extends AppCompatActivity {
//自定义控件
private NetworkImageView iv;

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    iv = (NetworkImageView) findViewById(R.id.imageView1);
    NetworkImageView();
}

public void NetworkImageView(){
    String ImageUrl = "http://i11.tietuku.com/10250403e075156a.jpg";
    RequestQueue requestQueue = Volley.newRequestQueue(this);
    final LruCache<String,Bitmap> lruCache = new LruCache<>(20);
    ImageLoader imageLoader = new ImageLoader(requestQueue, new ImageLoader.ImageCache() {
        @Override
        public Bitmap getBitmap(String s) {
            return lruCache.get(s);
        }

        @Override
        public void putBitmap(String s, Bitmap bitmap) {
            lruCache.put(s, bitmap);
        }
    });
    //默认加载图片资源
    iv.setDefaultImageResId(R.mipmap.ic_launcher);
    //发生错误时的图片资源
    iv.setErrorImageResId(R.mipmap.ic_launcher);
    iv.setImageUrl(ImageUrl,imageLoader);
}


}

这个也是非常简单，可分为五步：

  1. 创建一个RequestQueue对象。
  2. 创建一个ImageLoader对象。
  3. 在布局文件中添加一个NetworkImageView控件。
  4. 在代码中获取该控件的实例。
  5. 设置要加载的图片地址。
